# Слова из букв

Мобильная казуальная игра в жанре word-connect на русском языке. Из круга букв пользователь составляет слова, заполняя кроссворд.

## Запуск

Откройте `index.html` в браузере. Никакой сборки и установки не требуется.

> Если браузер ругается на ES-модули при открытии через `file://`, поднимите простой статический сервер:
> ```
> python -m http.server 8000
> ```
> и откройте `http://localhost:8000`.

## Структура

```
02_Words/
├── index.html
├── styles.css
├── js/
│   ├── main.js              # bootstrap
│   ├── config.js            # настройки баланса, рекламы, dev-режима
│   ├── dictionary.js        # ~2500 русских слов
│   ├── storage.js           # localStorage + миграции
│   ├── audio.js             # звуки через Web Audio API
│   ├── ads.js               # AdManager (mock + хуки под Яндекс)
│   ├── crossword.js         # рендер кроссворда
│   ├── levels.js            # 20 ручных уровней
│   ├── levels.generated.js  # 30 сгенерированных уровней
│   ├── levelGenerator.js    # автогенератор уровней
│   ├── input.js             # ввод (свайп по буквам)
│   ├── game.js              # логика игры
│   ├── ui.js                # экраны и кнопки
│   └── devPanel.js          # панель разработчика (?dev=1)
├── README.md
├── DEV_NOTES.md
└── TODO.md
```

## Dev-панель

Откройте `index.html?dev=1` чтобы получить панель генерации уровней.

## Дальше — APK

Игра задумана как обёртка для Android-приложения. Любой WebView-wrapper (Cordova / Capacitor / простой нативный WebView) подойдёт — все ресурсы загружаются с относительных путей и используют только localStorage.

Подробности — в `DEV_NOTES.md`.
