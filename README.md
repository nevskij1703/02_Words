# Слова из букв

Мобильная казуальная игра в жанре word-connect на русском языке. Из круга букв пользователь свайпом или кликами составляет слова и заполняет кроссворд. 50 встроенных уровней.

## Запуск

Открыть `index.html` в браузере.

> Современные браузеры запрещают ES-модули по `file://`. Если игра не стартует — поднимите простой статический сервер:
> ```
> python -m http.server 8000
> ```
> и откройте `http://localhost:8000`.

## Управление

- **Свайп или клики** по буквам в круге → составить слово.
- Слова кроссворда заполняются автоматически при их нахождении.
- Дополнительные слова из словаря, которые также можно составить, попадают в строку «бонусные».
- 💡 — подсказка (открыть случайную закрытую букву). Подсказки кончились — предлагается посмотреть рекламу.
- 🔀 — перемешать буквы.
- ✖ — сбросить текущий ввод.
- 🔊 / 🔇 — включить/выключить звук.

## Dev-панель

Откройте `index.html?dev=1`. Возможности:

- Сгенерировать уровень с произвольными параметрами.
- Прогнать `validateLevel()` на текущем уровне или на всех ручных (1-20).
- Сохранить уровень в localStorage / выгрузить JSON / загрузить JSON.
- Скачать накопленные уровни как `levels.generated.js` (готовый ES-модуль).

## Регенерация уровней 21-50

```
node scripts/generate_levels.mjs
```

Скрипт пробегает генератор, валидирует каждый уровень и перезаписывает `js/levels.generated.js`.
